**Hippie Hangman**

Hangman is a classic game made for word-lovers. Test your vocabulary and spelling skills! Hippie Hangman is based on words & phrases used in media in the late 1960's. 

**Technology Used**

* Javascript
* Vanilla JS
* HTML
* CSS3
